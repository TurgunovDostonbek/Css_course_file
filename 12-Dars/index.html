Z-index haqida!

Z-index - bu CSS xususiyati bo'lib, u elementlarning qatlamlarini boshqarish uchun ishlatiladi. Bu xususiyat, elementlarning qaysi biri ustun yoki pastda joylashishini belgilaydi. Z-index faqat pozitsiyalangan elementlar (position: relative, absolute, fixed yoki sticky) uchun amal qiladi.

Masalan, agar sizda ikkita element bo'lsa va birining z-index qiymati 1, ikkinchisining esa 2 bo'lsa, ikkinchi element birinchi element ustida ko'rsatiladi.

Overflow haqida!

Overflow - bu CSS xususiyati bo'lib, u elementning ichidagi kontentning qanday ko'rsatilishini boshqaradi. Agar kontent elementning o'lchamidan oshib ketsa, overflow xususiyati yordamida bu kontentni qanday ko'rsatishni belgilashingiz mumkin.

Masalan, agar sizda biror element bo'lsa va uning ichidagi kontent elementning o'lchamidan oshib ketsa, siz overflow: hidden; xususiyatini qo'llashingiz mumkin, bu holda kontent ko'rsatilmaydi. Yoki overflow: scroll; xususiyatini qo'llasangiz, element ichida skroll paneli paydo bo'ladi va foydalanuvchi kontentni ko'rishi mumkin.
