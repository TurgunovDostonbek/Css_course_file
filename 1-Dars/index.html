CSS Ummumiy tushunchalar

CSS (Cascading Style Sheets) - bu veb-sahifalarning ko'rinishini va tuzilishini boshqarish uchun ishlatiladigan uslub varaqalari tili. CSS yordamida siz HTML elementlariga rang, shrift, o'lcham, joylashuv va boshqa ko'plab uslublarni qo'llashingiz mumkin.

CSS versiyalari

  • CSS1: Birinchi versiya, 1996 yilda chiqarilgan.
  • CSS2: 1998 yilda chiqarilgan, ko'proq xususiyatlar va imkoniyatlar qo'shilgan.
  • CSS3: 2011 yilda chiqarilgan, modulga asoslangan yondashuvni taqdim etadi.

CSS sintaksisi

CSS sintaksisi quyidagi asosiy qismlardan iborat:

  1. Selektor: Uslub qo'llaniladigan HTML elementi.
  2. Xususiyat: O'zgartiriladigan elementning xususiyati (masalan, rang, o'lcham).
  3. Qiymat: Xususiyatga beriladigan qiymat (masalan, qizil, 16px).

CSS Comments

CSS izohlar /* va */ belgilar orasida yoziladi. Bu izohlar kodni o'qish va tushunishni osonlashtiradi.
