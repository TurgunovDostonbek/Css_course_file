Shorthands Qisqartmalar

CSS qisqartmalari - bu bir nechta CSS xususiyatlarini bitta xususiyatga birlashtirish imkonini beruvchi usul.

Misollar:

  • Margin qisqartmasi: margin: 10px 20px;
  • Padding qisqartmasi: padding: 5px 10px 15px;
  • Border qisqartmasi: border: 1px solid black;

HTML tree

HTML daraxti - bu HTML hujjatining tuzilishini ko'rsatadi.

Misollar:

  • Body: <body>
  • Div: <div>
  • Span: <span>

Multi classes

Bir element bir nechta klassga ega bo'lishi mumkin.

Misollar:

  • HTML: <div class="class1 class2">
  • CSS: .class1 {}
  • CSS: .class2 {}

!Important

CSS qoidalarini majburiy ravishda qo'llash uchun !important kalit so'zidan foydalanish mumkin.

Misollar:

  • CSS: color: red !important;
