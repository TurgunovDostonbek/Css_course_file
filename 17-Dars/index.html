Pseudo Class

Pseudo classlar - bu CSS da elementlarning holatini yoki o'zgarishini belgilash uchun ishlatiladigan maxsus tanlovchilar. Ular elementlarga qo'shimcha uslublar qo'llash imkonini beradi.

Pseudo Element
