Responsive dizayn

Responsive dizayn - bu veb-sahifalarning turli qurilmalarda, masalan, kompyuterlar, planshetlar va smartfonlarda yaxshi ko'rinishini ta'minlash uchun ishlatiladigan dizayn yondashuvi.

Media query

Media query - bu CSS xususiyati bo'lib, u veb-sahifaning ko'rinishini foydalanuvchining qurilmasiga mos ravishda o'zgartirish imkonini beradi.
